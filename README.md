# Homebrew Tap for Forge CLI

Homebrew formulae for [Forge](https://docs.candescent.com) — the Candescent developer CLI for building, submitting, and managing extensions.

## Install

```bash
brew tap candescent-dev/forge
brew install forge-cli
```

## Upgrade

```bash
brew update
brew upgrade forge-cli
```

## Uninstall

```bash
brew uninstall forge-cli
brew untap candescent-dev/forge
```

## About

Forge is a CLI that provides access to the Candescent Digital Insight APIs, widget and aspect development workflows, component submission, AI-powered docs search, and GitHub repository management.

- **Prerequisites:** macOS or Linux
- **Documentation:** https://docs.candescent.com
